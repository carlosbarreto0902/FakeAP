ℹ️ Sobre FakeAP

FakeAP es un sistema de seguridad que implementa un punto de acceso Wi-Fi trampa diseñado para atraer y controlar dispositivos que intentan conectarse a redes inalámbricas.

  • 🔧 Usa hostapd y dnsmasq para crear la red Wi-Fi y asignar direcciones IP dinámicas.
  • 🔐 Aplica filtrado MAC para restringir el acceso a dispositivos autorizados.
  • 📡 Captura y analiza el tráfico en tiempo real usando Scapy, permitiendo monitoreo detallado.
  • 🧠 Se integra con un dashboard web en Django que muestra dispositivos conectados, historial de tráfico y alertas de seguridad.
  • 🔁 Emplea reglas iptables para redireccionamiento inteligente, control de acceso y medidas de engaño.

Este sistema permite detectar intrusos en redes inalámbricas y contribuye al fortalecimiento de la ciberseguridad en entornos corporativos o académicos.
